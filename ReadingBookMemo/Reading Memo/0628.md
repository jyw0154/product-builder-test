### chapter 01
'IT 프로덕트의 기본 구조와 웹 개발 기초'

Front-End : 앞쪽, 사용자의 눈에 보이는 화면에서 작동하는 부분
클라이언트 사이드(Clinet-Side)라고도 한다.

Back-End : 뒤쪽, 사용자의 눈에는 보이지 않지만 프로덕트의 핵심 로직과 데이터 처리를 담당하는 부분을 말한다. Logic이란 프로그램이 어떻게 동작해야 하는 지를 정하는 논리적인 규칙들을 의미한다. 
주로 서버(Server)라고 불린다. 백엔드에서는 보안이 매우 중요한 부분.

웹 개발 ? 인터넷 브라우저를 통해 접속하여 이용할 수 있는 웹 서비스를 만드는 과정. URL로 접속하여 이용하는 모든 서비스가 *웹서비스*.

웹 개발은 접근성이 좋고, 만든 결과물을 전 세계 누구나 쉽게 사용할 수 있게 배포하기 용이한 장점이 있다.

##### 웹 개발 F/E 3가지 축.
1. HTML : 웹페이지의 구조와 내용을 담당하는 마크업 언어.
2. CSS : HTML로 만들어진 뼈대에 디자인을 입히는 스타일 시트 언어.
3. 자바스크립트 : 웹 페이지를 동작하게 하는 프로그래밍 언어.
프로그래밍 언어는 논리적인 사고와 복잡한 로직 설계를 요구, HTML은 단순히 문서의 구조를 정의하는 역할.

##### 라이브러리와 프레임워크
라이브러리가 개발자가 필요할 때 직접 골라서 꺼내 쓰는 '도구 모음', 프레임워크는 이미 정해진 틀과 규칙안에서 개발자가 필요한 부분만 채워 넣는 '작업 환경' 대표적인 라이브러리로 리액타가 있고, 대표적인 프레임워크로 넥스트Js가 있다.
